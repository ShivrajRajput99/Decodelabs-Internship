# Project-3: SQL Data Analysis Project

## Objective
Analyze e-commerce order data using SQL queries to extract meaningful business insights and understand customer purchasing behavior.

## Tools Used
- MySQL
- MySQL Workbench
- Microsoft Excel

## Dataset Information
- Dataset Type: E-commerce Orders Dataset
- Total Records: 1200

## SQL Concepts Used
- SELECT
- WHERE
- ORDER BY
- GROUP BY
- COUNT()
- SUM()
- AVG()

## Queries Performed
1. Display all records
2. Display specific columns
3. Find delivered orders
4. Find orders with quantity greater than 3
5. Sort orders by total price
6. Count total orders
7. Calculate total revenue
8. Calculate average order value
9. Count orders by status
10. Count orders by product
11. Calculate revenue by product
12. Analyze payment methods
13. Analyze referral sources
14. Find top 5 most expensive orders
15. Find average quantity per product

## Key Insights
- Analyzed order status distribution
- Calculated total revenue generated from sales
- Identified product-wise sales performance
- Examined customer payment preferences
- Evaluated referral source effectiveness

## Outcome
Generated insights about:
- Revenue performance
- Product popularity
- Customer purchasing trends
- Order status distribution
- Payment method usage

## Conclusion
This project demonstrates the use of SQL for data analysis on an e-commerce dataset using filtering, sorting, grouping, and aggregation functions.
